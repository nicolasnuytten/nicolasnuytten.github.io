Mobility

Week of Mobility

An informative website about events all about mobility in Belgium
Date: January 2018

watch online

Brief

This task was all about the week of mobility in Belgium. We had to make a website where visitors could see all of the events happening in that week near them or in a specific location (using AJAX). Also every event is stored in a database. Using PHP we had to show all of the difirent events.

Conclusion

After all this project was fun and nice, besides the strugles where I learend a lot off. Using PHP to display all the events and filter them. Also it needs to be instant (AJAX). The result is surely not my best but very happy what it became.
